So geht i.appear

User Guide

Short Guide

1. Rundgang oeffnen

Link oder QR-Code scannen. Kein App-Download noetig.

2. Station waehlen

Auf der Karte eine Station antippen oder einfach vor Ort starten.

3. Inhalt erleben

Lesen, hoeren, schauen – alles laeuft direkt im Browser.

4. Weiterziehen

Zur naechsten Station – bis alle Stationen erkundet sind.

Ausfuehrlicher Guide

Technik-Voraussetzungen

i.appear laeuft auf jedem aktuellen Smartphone mit Browser. Fuer AR-Features wird die Kamera benoetigt, fuer Audio-Features ein Kopfhoerer oder Lautsprecher. Eine stabile Internetverbindung macht das Erlebnis fluessiger.

Ohne App, ohne Konto

Es gibt bewusst keine App und keine Registrierung. Alles laeuft direkt im Browser – ohne Werbung, ohne Tracking, ohne Huerden.

Tipps vor Ort

Bei Sonnenlicht hilft ein Sonnenschutz fuer den Bildschirm. Fuer Audio-Inhalte empfehlen wir Kopfhoerer. Wer nicht vor Ort ist, kann viele Inhalte auch von zuhause aus erkunden.
